# Customer Purchase Prediction using Logistic Regression

## Project Overview

This project predicts whether a customer will purchase a product based on customer data such as age, salary, and other features. The model helps businesses understand customer behavior and improve marketing strategies.

## Objective

To build a machine learning model that can predict customer purchase decisions accurately.

## Dataset

The dataset contains customer information including:

* Age
* Estimated Salary
* Purchased (Target Variable)

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Machine Learning Algorithm

* Logistic Regression

## Project Workflow

1. Data Collection
2. Data Preprocessing
3. Handling Missing Values
4. Feature Scaling
5. Train-Test Split
6. Model Training using Logistic Regression
7. Prediction
8. Model Evaluation

## Evaluation Metrics

* Accuracy Score
* Confusion Matrix
* Classification Report

## Data Visualization

* Bar chart

## Result

The model successfully predicts customer purchase behavior with good accuracy.

## Conclusion

This project helped in understanding classification problems, data preprocessing, feature scaling, and logistic regression implementation.
